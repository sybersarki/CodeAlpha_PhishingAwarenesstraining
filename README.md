## Project Overview
An interactive, web-based Phishing Awareness Training module built as part of the CodeAlpha Cybersecurity Internship. This training educates users on how to identify, avoid, and report phishing attacks through real-world examples, visual demonstrations, and an interactive quiz.

## 🌐 Live Demo
[View Live Training Module](https://sybersarki.github.io/CodeAlpha_PhishingAwarenesstraining/)

## 📚 Training Modules

| Module | Topic |
|--------|-------|
| Module 01 | What is Phishing? |
| Module 02 | Types of Phishing Attacks |
| Module 03 | Recognizing a Phishing Email |
| Module 04 | How to Protect Yourself |
| Module 05 | Interactive Knowledge Quiz |

## ✨ Features
- **Interactive phishing email demo** — hover over red flags to learn why each element is suspicious
- **6 attack types covered** — Email, Spear, Whaling, Smishing, Vishing, Clone Phishing
- **5-question interactive quiz** — with instant feedback and scoring
- **Real-world statistics** — backed by industry data
- **8 best practices** — practical steps anyone can follow
- **Fully responsive** — works on desktop and mobile
- **No frameworks** — pure HTML, CSS, and JavaScript

## 🛡️ Topics Covered
- What phishing is and how it works
- Social engineering tactics used by attackers
- How to recognize phishing emails and fake websites
- Red flags to look for in suspicious messages
- Best practices to avoid falling victim
- Multi-Factor Authentication and why it matters
- How to report phishing attempts

## 🚀 How to Run
Simply open `index.html` in any modern web browser — no installation required.

```bash
git clone https://github.com/sybersarki/CodeAlpha_PhishingAwarenesstraining.git
cd CodeAlpha_PhishingAwarenesstraining
open index.html
```
